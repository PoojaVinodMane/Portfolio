This project is a modern, responsive personal portfolio built as a single-page application (SPA) to showcase skills, projects, education, and professional achievements in Computer Science and Engineering.

It features a dark theme accented with neon blue elements and custom JavaScript for dynamic interactivity.

---

### ‚ú® Features

* **Design & Theme:** Dark aesthetic with a striking neon blue color palette (`--primary: #00d4ff`).
* **Interactivity:** Neon/glow hover effects applied to content cards (Skills, Education, Certificates, POR, and Social Links).
* **3D Project Slider:** A custom-built JavaScript implementation featuring a 3D rotation effect for showcasing projects.
* **Responsive Layout:** Optimized for seamless viewing across all devices (desktop, tablet, mobile).
* **Animations:** Uses the Intersection Observer API for subtle, smooth fade-in and slide-up animations as content enters the viewport.
* **Dynamic Navbar:** Header changes styling (shrinks) upon scrolling.
